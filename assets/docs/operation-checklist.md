## 1. ローンチ前チェックリスト

### 1.1 Webflow設定

- [ ] すべてのページが作成されている（/、/how-to-enter、/login、/portal、/terms、/privacy、/legal）
- [ ] Navbarが全ページに表示されている
- [ ] Footerが全ページに表示されている
- [ ] リンクが正しく設定されている（404エラーがない）
- [ ] レスポンシブデザインが適切に動作する（モバイル・タブレット・デスクトップ）
- [ ] カスタムドメインが正しく設定されている
- [ ] SSL証明書が有効化されている（HTTPS）
- [ ] Webflow Hostingが本番環境に公開されている

### 1.2 Memberstack設定

- [ ] Memberstackスクリプトが Webflow Head Code に貼り付けられている
- [ ] Public Sign-up が OFF に設定されている（招待制）
- [ ] プラン「Free」（または入場権）が作成されている
- [ ] /portal ページが「Members only」に設定されている
- [ ] /login ページにログインフォームが表示されている（`data-ms-login-form`）
- [ ] Navbar の「ログイン」ボタンに `data-ms-hide="members"` が設定されている
- [ ] Navbar の「マイページ」ボタンに `data-ms-hide="non-members"` が設定されている

### 1.3 Typebot設定

- [ ] Typebotが作成されている（年末調整bot 2025年版）
- [ ] System Instructions が設定されている（税理士事務所提供のプロンプト）
- [ ] Knowledge Base に国税庁PDFがアップロードされている
- [ ] 入力検証（マイナンバー等の検出）が設定されている
- [ ] 埋め込みコードが /portal ページに貼り付けられている
- [ ] Typebot URLが正しく設定されている（`YOUR_TYPEBOT_URL` を置き換え済み）

### 1.4 SEO/メタタグ設定

- [ ] 各ページのTitle Tagが設定されている
- [ ] 各ページのMeta Descriptionが設定されている
- [ ] OGP（Open Graph）タグが設定されている
- [ ] OG画像（1200×630px）がアップロードされている
- [ ] robots.txt が適切に設定されている（/login と /portal は Disallow）
- [ ] sitemap.xml が生成されている
- [ ] Canonical URLが設定されている

### 1.5 法務ページ

- [ ] 利用規約が公開されている（/terms）
- [ ] プライバシーポリシーが公開されている（/privacy）
- [ ] 特商法表記が公開されている（/legal）
- [ ] 実際の事業者情報に置き換えられている（[ ] 内の項目）
- [ ] 弁護士または法務専門家のレビューを受けている

### 1.6 機能テスト

#### 非会員（未ログイン）

- [ ] ランディングページ（/）が正しく表示される
- [ ] /how-to-enter ページが正しく表示される
- [ ] /login ページが正しく表示される
- [ ] /portal にアクセスすると /login にリダイレクトされる
- [ ] Navbarの「ログイン」ボタンが表示される
- [ ] Navbarの「マイページ」ボタンが非表示になっている

#### 会員（ログイン後）

- [ ] 招待メールが正しく送信される
- [ ] 招待リンクからアカウント作成ができる
- [ ] ログインができる
- [ ] /portal にアクセスできる
- [ ] Typebotが正しく表示される
- [ ] Navbarの「マイページ」ボタンが表示される
- [ ] Navbarの「ログイン」ボタンが非表示になっている

#### Typebot機能テスト

- [ ] 質問に対して回答が返ってくる
- [ ] 回答に国税庁の参考リンクが付いている
- [ ] 確証レベル（【確証あり】【一般論】【仮説】）が明示されている

### 1.7 パフォーマンス

- [ ] ページの読み込み速度が3秒以内（Google PageSpeed Insights で確認）

   - 質問: 「[今年度の法改正内容]について教えてください」
   - 期待: 正確な回答 + 新年度の国税庁リンク

   - 期待: 不明または適切な質問の促し

## 4. トラブルシューティング

### 4.1 よくある問題と解決方法

#### 問題: ログイン後も /portal にアクセスできない

**原因**:
- Memberstackで /portal が「Members only」に設定されていない
- ユーザーに正しいプランが割り当てられていない

**解決策**:
1. Memberstack > Pages で /portal が「Members only」になっているか確認
2. Memberstack > Members でユーザーのプラン割り当てを確認
3. ユーザーに正しいプラン（Free）を割り当て

---

#### 問題: 招待メールが届かない

**原因**:
- メールアドレスの入力ミス
- 迷惑メールフォルダに振り分けられている
- Memberstackの送信エラー

**解決策**:
1. Memberstack > Members で招待ステータスを確認
2. 正しいメールアドレスに招待メールを再送信
3. ユーザーに迷惑メールフォルダの確認を依頼

---

#### 問題: Typebotが表示されない

**原因**:
- iframe の src URLが間違っている
- Typebotが公開されていない
- ブラウザの設定で iframe がブロックされている

**解決策**:
1. /portal ページの Embed コードで iframe src を確認
2. Typebot > Share で公開設定を確認
3. 別のブラウザでテスト

---

#### 問題: Typebotの回答に参考リンクが付かない

**原因**:
- System Instructions に「参考リンクを記載」の指示がない
- Knowledge Base の設定が不正確

**解決策**:
1. System Instructions を確認し、参考リンク記載の指示を追加
2. Knowledge Base の Priority を「High」に設定
3. Auto-reference を ON に設定

---

### 4.2 緊急時の対応

#### サイト全体がダウンした場合

1. Webflowのステータスページを確認: https://status.webflow.com/
2. カスタムドメインのDNS設定を確認
3. Webflow サポートに連絡

#### Memberstackが機能しない場合

1. Memberstackのステータスページを確認: https://status.memberstack.com/
2. Memberstack スクリプトが正しく読み込まれているか確認（ブラウザのDevTools）
3. Memberstack サポートに連絡

#### Typebotが機能しない場合

1. Typebotのステータスページを確認
2. iframe の src URLを確認
3. Typebot サポートに連絡

## 7. バックアップとリカバリー

### 7.1 バックアップ対象

- Webflowプロジェクト（自動バックアップ）
- Memberstack会員データ（エクスポート機能を使用）
- Typebotの設定（スクリーンショットまたはエクスポート）
- 法務ページのテキスト（ローカルに保存）

### 7.2 リカバリー手順

1. Webflowプロジェクトの復元: Webflow > Site Settings > Backups
2. Memberstack会員データのインポート: Memberstack > Members > Import
3. Typebotの再設定: Knowledge BaseとSystem Instructionsを再入力

## 8. ユーザーサポート

### 8.1 よくある質問（FAQ）

**Q1. 招待メールが届きません**
A. 迷惑メールフォルダをご確認ください。それでも届かない場合は、support@yourdomain.com までご連絡ください。

**Q2. パスワードを忘れました**
A. /login ページの「パスワードを忘れた方」リンクからリセットできます。

**Q3. 利用期間の延長はできますか？**
A. 延長をご希望の場合は、事務局までご連絡ください。

### 8.2 サポート対応時間

- 平日: 10:00〜17:00
- 土日祝日: 対応なし
- 返信期限: 3営業日以内
